Common questions

What is Continuous Protection?
Grassion watches your repo 24/7. Every commit triggers an automatic scan. If a new unprotected endpoint appears, Grassion raises a fix PR immediately.
What frameworks does Grassion support?
Currently optimized for Node.js/Express. Django, FastAPI, Rails, and Spring Boot support coming soon.
Is my code stored on your servers?
No. Grassion reads your code temporarily to analyze it but never stores source code. Only scan metadata is retained.
Can I cancel anytime?
Yes. Cancel anytime from your dashboard. Pro features remain active until the end of your billing period.
What payment methods are accepted?
All major credit/debit cards, UPI, net banking, and wallets through Razorpay.
Does Grassion work with private repos?
Yes. Grassion works with both public and private repositories through GitHub OAuth.
