How to Add OpenAI API Key

1. Go to platform.openai.com → API Keys → Create new secret key
2. Copy the key (starts with sk-)
3. In Railway: go to your app service → Variables → Add variable:
OPENAI_API_KEY = sk-your-key-here
4. Redeploy. AI summaries and Impact Intelligence will start working immediately.
